# MultipassTechnicalTest
This is my submission for Canonical's Multipass engineer technical test.

## Installation
This project requires the installation of two libraries, `nlohmann` and `libcurl`.

### macOS
Using Homebrew
1. Install `nlohmann`
```sh
   brew install nlohmann-json
```
2. Install `libcurl`
```sh
   brew install curl
```
